Installer with binaries created using [[Macports][<macports:>]].
Installs =|/opt/local/bin/swipl|=. Needs [[xquartz][<xquartz:>]] (X11)
installed for running the [[development tools][</IDE.html>]]. Currently,
version 2.7.4 is required. You can check the version by opening an X11
application and then checking `about' in the X11 menu. This binary is
linked against GMP 5, which implies that it is covered by the LGPL-V3
license. See below.
